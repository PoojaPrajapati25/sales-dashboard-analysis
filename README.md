# Sales Dashboard Analysis

This project is an interactive Power BI dashboard created using a sales dataset from Excel.

## Dashboard Features

* Total Sales KPI
* Monthly Sales Trends
* Sales by Category
* Regional Sales Analysis
* Sales by Sub-Category
* Interactive Filters

## Tools Used

* Power BI
* Excel Dataset
* Data Visualization
* Business Intelligence

## Key Insights

* Technology category generated the highest sales.
* Sales increased significantly during November and December.
* Phones and Chairs were among the top-performing sub-categories.

## Purpose

The purpose of this project is to analyze sales performance and present business insights using interactive visualizations.
